## Citation Format Reference

### Single Author
```
[Author, Year]
Example: [Isla, 2005]
```

### Two Authors
```
[Author1 & Author2, Year]
Example: [Millington & Funge, 2009]
```

### Three or More Authors
```
[First Author et al., Year]
Example: [Lewis et al., 2020]
```

### Multiple Citations
```
[Author1, Year1; Author2, Year2]
Example: [Isla, 2005; Orkin, 2006]
```
